# 冬夜烟花表白网站

一个浪漫的单页表白网站，主题为"冬夜烟花表白"，专为在立冬这天向女朋友朱奕霖表白而设计。

## 🌟 特色功能

### ✨ 视觉效果
- **完美圆形烟花爆炸**：点击屏幕任意位置绽放出圆形烟花
- **动态文字显示**："朱奕霖"在烟花中心闪现，文字颜色会随机变化与背景和谐融合
- **梦幻星空背景**：150颗闪烁星星模拟真实夜空
- **飘雪氛围**：持续的雪花飘落效果
- **渐变背景**：深蓝夜空渐变，轻微动态变化

### 💕 互动体验
- **点击触发烟花**：每点击一次屏幕产生一朵烟花和随机情话
- **10句温柔情话**：随机显示关心和浪漫的话语
- **特别告白动画**：第5次点击后触发温馨的告白场景
- **飘浮爱心效果**：告白时爱心在屏幕中飘浮
- **音乐控制**：支持背景音乐播放控制

### 📱 技术特点
- **响应式设计**：完美适配手机、平板和电脑
- **Canvas动画**：流畅的烟花和粒子效果
- **优雅过渡**：所有动画都经过精心设计
- **无障碍支持**：支持减少动画偏好设置

## 🎮 使用方法

1. **打开网站**：在浏览器中打开 `fireworks_love.html`
2. **观看引导**：页面中央会显示"点击屏幕任意位置观看烟花"
3. **点击互动**：点击屏幕任意位置触发烟花效果
4. **观察情话**：每次点击后随机显示温柔情话
5. **等待惊喜**：点击5次后触发特别的告白动画

## 🎵 添加音乐

如需添加背景音乐，请：

1. 准备一段轻柔的背景音乐文件（推荐钢琴曲）
2. 在HTML文件中找到 `<audio id="backgroundMusic">` 标签
3. 将音乐文件路径填入 `src` 属性
4. 推荐音乐：
   - 《River Flows in You》
   - 《Kiss the Rain》
   - 轻柔的冬日氛围音乐

## 🎨 定制说明

如需修改网站内容，可以调整：

### 文字内容
- 修改 `sweetMessages` 数组中的情话
- 修改告白色中的文字内容
- 调整页脚文字

### 视觉效果
- 修改烟花颜色数组 `fireworkColors`
- 调整粒子数量和生命期
- 修改背景渐变色

### 互动设置
- 调整触发告白的点击次数
- 修改动画持续时间
- 调整音乐控制设置

## 💡 使用建议

- **最佳时机**：立冬当天或浪漫的冬夜
- **分享方式**：将文件发送给朱奕霖，让她用手机或电脑打开
- **互动引导**：告诉她点击屏幕5次有特别惊喜
- **环境营造**：建议在安静的环境中使用，配上轻柔的背景音乐
